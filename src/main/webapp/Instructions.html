THIS SOFTWARE IS PROVIDED "AS IS" AND ANY EXPRESSED OR IMPLIED WARRANTIES, INCLUDING, BUT NOT LIMITED TO, THE IMPLIED WARRANTIES OF MERCHANTABILITY AND FITNESS FOR A PARTICULAR PURPOSE ARE DISCLAIMED. See the full public license text.

 
Instructions:
  1. Extract the contents of the zip file to a temporary directory.
  2. Install the MCQuiz.class file into the servlets directory of your servlet enabled web server.
  3. Add the MCQuiz.jar file to the classpath for your servlet engine.
  4. Modify your startup configuration in the properties file for your servlet engine to indicate the path where the quiz.dtd file and all ".quiz" files will reside. See the servlet.properties file included with the distribution.
  5. Download and install the XML parser from IBM Alpha Works - XML4J. (See link below) Make it available to your servlet engine class path.
  6. Use the sample World Geography quiz supplied in the distribution. You may also create your own multiple choice quizes that conform to the Document Type Definition provided in the quiz.dtd file.
